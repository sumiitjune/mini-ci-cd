# Mini CI/CD Pipeline

A simple CI/CD system using Flask, Docker, GitHub Webhooks, and ngrok.

## 🔧 Features
- GitHub webhook integration
- Auto deployment trigger
- Docker-based backend
- Public exposure via ngrok

## 🛠 Tech Stack
- Python (Flask)
- Docker
- GitHub Webhooks
- ngrok

## 🚀 How it works
Push → Webhook → Flask → Deployment
